A **JR Hortifruti Delivery** é uma distribuidora/varejista de hortifruti com foco em venda de frutas, legumes, verduras, raízes e itens frescos por atendimento direto.

A presença digital deve comunicar:

- Confiança.

- Qualidade visual superior ao padrão informal de redes sociais.

## 4. Filosofia Visual

A direção visual recomendada segue o conceito **Colheita Urbana**.

Esse conceito une natureza, cidade e curadoria. A marca não deve parecer apenas um hortifruti de bairro comum; deve transmitir frescor, organização, qualidade e confiança.

### Princípios visuais

- Fundo claro/off-white.
- Verde profundo como cor institucional.
- Dourado/ocre para acabamento premium.
- Laranja como acento de energia e produto maduro.
- Fotos reais dos produtos como prova de autenticidade.
- Formas orgânicas combinadas com grid limpo.
- Espaço negativo para transmitir organização.
- Botões claros e objetivos.
- Hierarquia visual forte.

## 16. Performance

### Regras

- Converter imagens para WebP/AVIF.
- Usar lazy loading em imagens abaixo da primeira dobra.
- Definir width e height em imagens.
- Evitar bibliotecas pesadas.
- Não usar carrossel pesado na primeira versão.
- Não carregar vídeos automaticamente.
- Evitar animações complexas em mobile.
- Priorizar HTML, CSS e React simples.

### Primeira versão ideal

- React + TypeScript + Vite.
- Tailwind CSS.
- Sem dependências extras desnecessárias.
- WhatsApp via link.
- Catálogo estático em arquivo `products.ts`.
